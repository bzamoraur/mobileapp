# H0 prospect list — Spanish boutique travel advisors

*A researched starting list of **~44 real, ICP-fit** Spanish boutique/bespoke travel advisors & small agencies (honeymoons, safaris, luxury tailor-made). Compiled from web research (June 2026) with a source per row. Use it to run §5 of `docs/GTM-H0.md`.*

> **Verify before you send.** Rows were verified from indexed search results (the research couldn't load some sites/Instagram directly). Treat names/handles as real but **confirm the contact + that they're still active/ICP on a 10-second site visit** before outreach. Where a public email wasn't found, the channel is the web form or an Instagram DM.
>
> **Outreach compliance (Spain/EU, light touch):** prefer **personalized 1:1** messages to **public business** contacts (web form, info@, IG/LinkedIn DM) — not bulk cold-email blasts (Spain's LSSI-CE restricts unsolicited bulk commercial email). Identify yourself, make it relevant, offer an easy "no", and stop if asked. A free branded demo of *their* trip is both the best opener and the most welcome.

## How to work this list
1. **Verify** the contact + ICP on each site (10s). Drop any that aren't a fit.
2. **Tier 1 first:** build a **free Trojan-horse demo** of their kind of trip (safari → Tanzania link; honeymoon/luxury → Maldivas link) and open with it (scripts in `docs/GTM-H0.md` §6).
3. **Track** touches → replies → calls → pilots in your CRM/sheet (import `prospects.csv`). Aim ~40 touches → ~10 calls → 3 paid.
4. Add **Rodrigo (Pangea)** + any personal-network advisors at the top — warm always wins.

*Excluded by design (too large / not boutique-independent): Pangea, Nuba, Bru & Bru, Madrid & Beyond, Jacada, Totally Spain. Add back if you want to test enterprise-leaning targets.*
